# 쇼피파이 메타 오브젝트 & Matrixify 완벽 가이드

> 데이터 구조 설계부터 대량 등록까지 한 번에 배우는 실무 가이드

## 📖 소개

쇼피파이 스토어의 데이터 확장성을 위해 **메타 오브젝트(Metaobjects)**를 도입하고, **Matrixify** 앱을 활용하여 수천 건의 데이터를 효율적으로 처리하는 방법을 배웁니다.

## 🎯 학습 목표

- ✅ 쇼피파이 메타 오브젝트 개념 이해
- ✅ 커스텀 데이터 구조 설계 방법
- ✅ Matrixify를 활용한 대량 데이터 처리
- ✅ CSV 포맷 작성 및 검증
- ✅ 실전 에러 트러블슈팅

## 🚀 시작하기

브라우저에서 `index.html` 파일을 열면 인터랙티브 가이드가 시작됩니다.

```bash
# 로컬에서 실행
open index.html

# 또는 HTTP 서버 실행
python -m http.server 8000
# http://localhost:8000 접속
```

## 📚 주요 내용

### 1. 메타 오브젝트란?
- 쇼피파이 기본 구조를 확장하는 커스텀 데이터 타입
- 독립적인 데이터 구조 생성 및 관리
- 상품과 연결하여 활용

### 2. 데이터 구조 설계
- 필드 타입 선택 (Text, Rich text, File, URL 등)
- Key(변수명) 명명 규칙
- 필수/선택 필드 구분

### 3. Matrixify 워크플로우
- **Step 1: Export** - 템플릿 확보
- **Step 2: Preparation** - 데이터 가공
- **Step 3: Import** - 업로드 및 적용

### 4. 트러블슈팅
- CSV 파싱 에러 (쉼표 문제)
- Definition 누락 에러
- Handle 중복 에러

## 💡 주요 기능

### CSV 검증기
실시간으로 CSV 데이터를 검증하여 에러를 미리 발견할 수 있습니다.

- 필수 컬럼 체크
- 컬럼 개수 일치 확인
- Handle 중복 검사
- 쉼표 문제 감지
